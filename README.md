# Sonya Pen Chooser

Программа на Python для выбора ручки по количеству слов.

## Функциональность

- Ввод количества слов
- Проверка кратности числа
- Выбор ручки:
  - Если число кратно 5: "Соня пишет пятью ручками"
  - Если число кратно 3: "Соня пишет тремя ручками"
  - Во всех остальных случаях: "Соня пишет ярко-зеленой ручкой"

## Пример работы

Введите количество слов: 10
Соня пишет пятью ручками

Введите количество слов: 9
Соня пишет тремя ручками

Введите количество слов: 7
Соня пишет ярко-зеленой ручкой

## Куда можно встроить

- В игровые механики с выбором инструментов
- В обучающие материалы по оператору %
- Как пример множественного выбора

## Требования

- Python 3.x

## Примечание

Проверка на кратность 5 выполняется раньше проверки на кратность 3. Числа, кратные одновременно 3 и 5 (например, 15), попадут в первую ветку.
